# Code of Conduct

Dette prosjektet følger Contributor Covenant Code of Conduct.

## Vårt løfte

Vi forplikter oss til å gjøre deltakelse i prosjektet til en opplevelse fri for trakassering,
uansett alder, kroppsstørrelse, synlig/ikke-synlig funksjonsvariasjon, etnisitet,
kjønnsidentitet og uttrykk, erfaring, utdanning, sosioøkonomisk status, nasjonalitet,
utseende, rase, religion eller seksuell identitet og orientering.

## Vår standard

Eksempler på atferd som bidrar til et positivt miljø:

- Vær respektfull og inkluderende
- Ta imot konstruktiv feedback
- Fokuser på hva som er best for fellesskapet

Uakseptabel atferd inkluderer trakassering, fornærmelser og personangrep.

## Håndheving

Rapporter uakseptabel atferd ved å kontakte maintainerne via GitHub Issues (bruk en privat kanal hvis nødvendig).

Basert på Contributor Covenant v2.1: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
