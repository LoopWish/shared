# Contributing to Loopwish

Takk for at du vil bidra!

## Før du starter

- Vær høflig og konstruktiv (se [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)).
- Søk i eksisterende Issues/PR-er før du lager nye.

## Hvordan bidra

1. Fork repoet og lag en feature-branch: `feature/<kort-beskrivelse>`.
2. Gjør endringene små og fokuserte.
3. Oppdater dokumentasjon når relevant.
4. Legg til/oppdater tester der det finnes testoppsett.
5. Kjør lokale sjekker før PR (lint/test/build når aktuelt).

## Commit-meldinger

Bruk gjerne Conventional Commits:

- `feat: ...`
- `fix: ...`
- `refactor: ...`
- `docs: ...`
- `chore: ...`

## Pull Requests

- Fyll ut PR-templaten.
- Beskriv hva og hvorfor.
- Link til relevante issues.

## Rapportere bugs / foreslå features

Bruk issue-templatene i `.github/ISSUE_TEMPLATE/`.
